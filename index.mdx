<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get your WhatsApp automation up and running in minutes</p>
  
  <Steps>
    <Step title="Set up n8n Environment">
      Install n8n and configure your workspace with the required credentials for WhatsApp Business API, PostgreSQL, MongoDB, and Redis.
      
      <Note>
        You'll need a Meta Business account with WhatsApp Business API access to proceed.
      </Note>
    </Step>
    
    <Step title="Import Workflows">
      Import the two core workflows into your n8n instance:
      - **Receptor**: Handles incoming WhatsApp messages and buffers them in Redis
      - **Procesador**: Processes buffered messages with AI agents and manages business logic
      
      ```bash
      # Import from the workflows directory
      n8n import:workflow --input="workflows/Picantería Lurwis _ Receptor.json"
      n8n import:workflow --input="workflows/Picantería Lurwis _ Procesador.json"
      ```
    </Step>
    
    <Step title="Configure Webhook">
      Set up the Meta webhook URL to point to your Receptor workflow endpoint:
      
      ```
      https://your-domain.com/webhook/meta-verify
      ```
      
      Configure webhook verification and activate the workflow to start receiving messages.
    </Step>
    
    <Step title="Test the System">
      Send a test message to your WhatsApp Business number and watch the automation handle it:
      
      <Accordion title="Example test conversation">
        **You**: "Hola, quiero hacer un pedido"  
        **Bot**: "¡Bienvenido a Picantería Lurwis! Estoy aquí para ayudarte con tu pedido. ¿Qué te gustaría ordenar hoy?"
      </Accordion>
    </Step>
  </Steps>
</div>
